# Kioku - 暗記カードアプリ

スワイプ操作で快適に学習できるモバイルファーストの暗記カードアプリ。

## 使い方

**カード学習**

- カードをタップ → 答えを表示
- 右スワイプ → わかった（習得）
- 左スワイプ → わからない（復習）

**フィルタ・管理**

- カテゴリ別の絞り込み
- 学習状態（未学習 / 復習 / 習得）でフィルタ
- シャッフル・進捗リセット
- 一覧表示で全体を確認

学習の進捗はブラウザのローカルストレージに自動保存されます。

## 技術スタック

- React 19 + TypeScript
- Vite
- Tailwind CSS

## セットアップ

```bash
npm install
npm run dev
```

`http://localhost:5173` で起動します。

## ビルド

```bash
npm run build
```

`dist/` に出力されます。
